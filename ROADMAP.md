## 💡 Идеи / бэклог
- **Удаление планов с подтверждением** — свайп/кнопка удаления на строке плана + конфирм.
- **Общее время готовки плана** — показывать суммарное время на все блюда
  (сколько займёт наготовить весь план за раз).
- **Отмеченные покупки — вниз группы** — когда отмечаешь пункт в списке покупок,
  он уезжает в конец своей категории (купленное отдельно от невыбранного).
- **Убрать старые спиннеры** — где-то в приложении ещё остались `.spinner` (крутилки);
  заменить на кулинарный лоадер `ew-cooking` или убрать.
- **Tool calling для модели** — дать модели инструменты, чтобы она сама запрашивала
  данные: получить список уже существующих планов, искать по ним (по блюдам,
  ингредиентам, неделям), тянуть принятые/отклонённые. Тогда диалог сможет
  опираться на реальную историю, а не только на подмешанный в промпт список.
- **Валидатор реалистичности названий** — ловить «Голубая свинина», «Синий суп» и т.п.
- **Валидатор количеств** — ловить абсурд вроде «форель 4 кг на 2 порции».
- **Дашборд расхода токенов** — где смотреть, сколько съедено по моделям.

## ✅ Сделано
- **Единый лейаут по гайдбуку** — все экраны (Главная, Чат, Планы, План, Покупки, Печать, Блюдо)
  на каркасе `.page`: единый ритм, шапки одной высоты, фикс/не-фикс совпадают. Раздел в GUIDEBOOK.md.
- **Шеринг PDF-файлом** — страница печати отдаёт PDF (бэкенд fpdf2 + DejaVu), кнопка «Поделиться»
  кидает файл в системный шэр (Телеграм и т.п.); тумблеры Рецепты/Покупки с чекбоксами.
- **Секции на «Планах»** — Принятые (последние 3 + «Загрузить ещё» +10) и История (отклонённые+черновики).
- **Даты на следующую неделю** — план всегда на ближайшую неделю (готовка на выходных перед ней).
- **Потоковая генерация плана** (SSE): блюда прилетают по одному, лоадер внутри карточки,
  автоскролл, кнопки только по готовности. Тот же вызов DeepSeek — токенов не больше.
- **Потоковый список покупок** (SSE) из детерминированной базы — мгновенно и точно
  (модель в потоке теряла числа, поэтому убрана из этого пути).
- **Деплой на Pi** (192.168.1.230:8080) + репозиторий на GitHub (khripunovpp/easy-week);
  Пай теперь тянет обновления через git (deploy/update.sh).
- Кулинарный лоадер (кастрюлька с паром) + пул ~100 фраз вместо спиннеров.
- Метки в логах: какой запрос уходит в какой API (провайдер · модель · назначение).
- DeepSeek для плана + короткие шаги; развёрнутые инструкции и список — Cloudflare.
- Список покупок в граммах у источника (без хаоса единиц).
- Локальная разработка в Docker с живыми логами; SW погашен в dev-сборке.
- Экран приветствия, выбор кол-ва блюд, фейд-блюр шапок/футеров, вью плана.
